# Battle-2048
识别了用户请求描述游戏的意图。Battle 2048 是一个双人对战版的 2048，两个玩家各自操控一块 4×4 的棋盘，同屏竞技。 核心玩法和原版 2048 一样——滑动方向键让数字格合并，相同数字碰到一起就翻倍，目标是叠出尽可能高的数字、积累尽可能多的分数。谁棋盘先满到无法移动，谁就输。 不过这个版本在标准 2048 的基础上加了两个新机制。 第一个是行消除。当某一行四格全部填满时，整行会被清除并获得额外积分，有点像俄罗斯方块的思路——这让你在快满盘的危险时刻反而可能解套。 第二个是技能系统。棋盘上会随机刷出带 emoji 的技能格，把它们滑到棋盘边缘或数字格旁边就能收集，消除满行时也会顺手带走其中的技能格。每人最多存三个技能
